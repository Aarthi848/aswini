## 🛠️ Core Capabilities

### 🖥️ Mainframe Modernization
- **SCM Migration** — Endevor → Git transition with discovery workshops & tailored roadmaps

- **Application Refactoring** — IBM WatsonX Code Assistant for COBOL → Java migration
- **AIOps & Observability** — Real-time monitoring, distributed tracing, root cause analysis
